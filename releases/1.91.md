# Release Notes — Boost 1.91.0

Released: 2026-04-22

## Headers Only Libraries

- [libbitcoin-boost 1.91.0](https://nuget.org/packages/libbitcoin-boost/1.91.0)

## Precompiled Libraries

- [libbitcoin-boost-vc145 1.91.0](https://nuget.org/packages/libbitcoin-boost-vc145/1.91.0) — Visual Studio 2026 (MSVC 14.5 / v145)

## Notes

- First **stable** vc145 release. Previous `1.91.0-head` packages (built before the official
  Boost release) are superseded by this version.
- Boost 1.91 ships native vc145 support — no `msvc.jam` patching required (unlike the
  vc1450 workaround used for Boost 1.90).
- The `vc1450` toolset key (`libbitcoin-boost-vc1450`, Boost 1.90 only) is distinct from
  `vc145` and will not be updated further.
