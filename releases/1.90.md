# Release Notes — Boost 1.90.0

## Headers Only Libraries

- [libbitcoin-boost 1.90.0](https://nuget.org/packages/libbitcoin-boost/1.90.0)

## Precompiled Libraries

- [libbitcoin-boost-vc143 1.90.0](https://nuget.org/packages/libbitcoin-boost-vc143/1.90.0) — Visual Studio 2022 (MSVC 14.3x)
- [libbitcoin-boost-vc1450 1.90.0](https://nuget.org/packages/libbitcoin-boost-vc1450/1.90.0) — Visual Studio 2026 (MSVC 14.50, pre-release patch)

## Notes

- First release under the `libbitcoin-boost` package family (prior packages used the
  `boost` prefix from the original sergey-shandar/getboost maintainer).
- `vc1450` required a manual patch to `msvc.jam` in the Boost 1.90 source tree to
  recognise toolset `14.50`. See [CHANGES.md](../CHANGES.md) for details.
- Boost 1.91+ supersedes this release for VS2026 users — use `vc145` packages instead.
